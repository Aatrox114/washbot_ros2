# 洗车机器人底盘串口通信协议 v0.1

## 1. 通信目标

上位机 ROS2 节点负责订阅 `/cmd_vel`，通过串口向下位机发送速度控制指令。

下位机负责接收速度指令，完成电机控制、编码器积分，并通过串口返回里程计数据。

上位机收到里程计数据后，发布：

- `/odom`
- `odom -> base_link` TF
- `/base_status`

---

## 2. 基本约定

| 项目 | 内容 |
|---|---|
| 字节序 | 小端 |
| 校验 | CRC16-MODBUS |
| CRC 初值 | 0xFFFF |
| CRC 多项式 | 0xA001 |
| 帧头 | 0xAA 0x55 |
| 默认波特率 | 115200 |
| 上位机发送频率 | 50 Hz |
| 超时保护 | 0.5 秒无速度帧，下位机应停车 |

---

## 3. 上位机到下位机：速度控制帧

帧类型：`0x01`

| 字段 | 长度 | 类型 | 单位 | 说明 |
|---|---:|---|---|---|
| head | 2 | uint8[2] | - | 0xAA 0x55 |
| type | 1 | uint8 | - | 0x01 |
| len | 1 | uint8 | byte | 0x07 |
| seq | 1 | uint8 | - | 序号 |
| vx | 2 | int16 | mm/s | 前后速度 |
| vy | 2 | int16 | mm/s | 左右速度，差速底盘可为 0 |
| wz | 2 | int16 | mrad/s | 角速度 |
| crc | 2 | uint16 | - | CRC16-MODBUS，小端 |

总格式：

text
AA 55 | 01 | 07 | seq | vx_L vx_H | vy_L vy_H | wz_L wz_H | crc_L crc_H

## 4. 下位机到上位机：里程计反馈帧
| 字段     | 长度 | 类型       | 单位     | 说明              |
| ------ | -: | -------- | ------ | --------------- |
| head   |  2 | uint8[2] | -      | 0xAA 0x55       |
| type   |  1 | uint8    | -      | 0x02            |
| len    |  1 | uint8    | byte   | 0x14，也就是 20     |
| seq    |  1 | uint8    | -      | 可回传最近一次速度帧序号    |
| x      |  4 | int32    | mm     | odom 坐标系下 x     |
| y      |  4 | int32    | mm     | odom 坐标系下 y     |
| yaw    |  4 | int32    | mrad   | 车体航向角           |
| vx     |  2 | int16    | mm/s   | 当前 x 方向速度       |
| vy     |  2 | int16    | mm/s   | 当前 y 方向速度       |
| wz     |  2 | int16    | mrad/s | 当前角速度           |
| status |  1 | uint8    | -      | 状态字节            |
| crc    |  2 | uint16   | -      | CRC16-MODBUS，小端 |

## 5. 状态字节建议
|       bit | 含义    |
| --------: | ----- |
|      bit0 | 急停状态  |
|      bit1 | 电机错误  |
|      bit2 | 编码器错误 |
|      bit3 | 低电压   |
|      bit4 | 通信超时  |
| bit5-bit7 | 保留    |
